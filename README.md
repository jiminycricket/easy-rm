# EASY RM
[![GitHub issues](https://img.shields.io/github/issues/jiminycricket/easy-rm.svg)](https://github.com/jiminycricket/easy-rm/issues)

## Install

npm:
```
npm install -save easy-rm
```

yarn:
```
yarn add easy-rm
```

## Usage

```
const rm = require('./index.js');
rm('/test/test.txt')

```

## test

```
npm run test
```
